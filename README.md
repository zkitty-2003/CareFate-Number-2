# CareFate Number 2

Web application for tracking health and fate.

## How to Run

1. Double-click `run.bat` to start the server.
   (This will automatically install necessary libraries like `fastapi` and `uvicorn`)

2. The browser should open automatically to `http://localhost:8000/index.html`.

## Development

- Frontend: HTML/CSS/JS (Vanilla)
- Backend: FastAPI (`main.py`)
- Database: Supabase
